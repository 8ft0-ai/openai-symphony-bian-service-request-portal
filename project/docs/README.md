# Project Docs

This folder holds project-level decisions and operating processes that should
not live inside the ticket import assets themselves.

## Index

- [Kanban and Linear operating model](./decisions/001-kanban-linear-operating-model.md)
- [Linear import runbook](./processes/linear-import-runbook.md)

## Scope

Use `project/docs` for:

- delivery decisions that affect how the team will run the project
- import and planning runbooks
- workflow rules that need to stay stable while the implementation backlog moves

Do not use `project/docs` for:

- raw Linear export files
- generated CSV artifacts
- ticket source markdown
